# Mineração de Dados

Este diretório contém materiais e atividades relacionadas à disciplina de Mineração de Dados. Ele abrange desde os conceitos fundamentais até as etapas práticas do pipeline de mineração de dados, incluindo KDD, limpeza, pré-processamento e transformação de dados.

**Etapas-pipeline**: Sequência estruturada que transforma dados brutos em conhecimento através de coleta, limpeza, transformação, mineração e avaliação.

**KDD**: Processo completo de Descoberta de Conhecimento que integra seleção, pré-processamento, transformação, data mining e interpretação de padrões.

**Limpeza-dados**: Técnicas para tratar dados ausentes, duplicados, ruidosos e inconsistentes usando imputação, remoção ou interpolação.

**Pré-processamento**: Operações preparatórias como normalização, codificação categórica, remoção outliers e balanceamento antes da mineração.

**Transformacao-dados**: Engenharia de atributos criando novas features, PCA, normalização e agregações para revelar padrões ocultos.

**Árvore-de-decisão**: Algoritmo supervisionado que gera nós de decisão, ramos condicionais e folhas classificadoras por impureza (Gini/Entropia).

**Regressão**: Supervisionada que prediz valores contínuos minimizando erro quadrático (R², RMSE).

**Clusterização**: Não-supervisionada que agrupa dados similares (K-Means, silhueta).

**Classificação**: Supervisionada que atribui categorias (F1, matriz confusão).

**AutoML**: Automatiza pipeline testando algoritmos/hiperparâmetros (PyCaret, AutoGluon).

**Métricas**: Indicadores por problema (R² regressão, F1 classificação, lift regras).

**Regras-associação**: Não-supervisionada que encontra X→Y em transações (Apriori, suporte/confiança).

**Validação-cruzada**: Divide dados K folds testando Kx (treina K-1, testa 1) para média±desvio confiável.
