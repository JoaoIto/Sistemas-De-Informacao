# Gabarito Comentado

**1. Resposta Correta: C (Média > Mediana > Moda)**
* **Por que?** Quando temos valores atípicos muito altos (os usuários que deixam o sistema aberto o dia todo), eles "puxam" a Média para cima. A Moda fica presa no valor mais comum (10 minutos) e a Mediana fica no meio. Se $As > 0$ (Assimetria Positiva ou à direita), a ordem é sempre Média > Mediana > Moda.

**2. Resposta Correta: B (Leptocúrtica; mais pontiaguda...)**
* **Por que?** O valor de referência da Curtose para uma curva normal (Mesocúrtica) é $K = 0,263$. Se o valor encontrado é menor que a base ($0,18 < 0,263$), a distribuição é **Leptocúrtica**. O prefixo "lepto" lembra algo fino/estreito. Significa que há um pico muito alto, com as idades dos alunos muito concentradas perto da média.

**3. Resposta Correta: C (Não existe correlação linear...)**
* **Por que?** Essa é uma das pegadinhas mais clássicas. Quando $r = 0$, nós dizemos que existe "Correlação Nula". Isso significa especificamente que **não há relação linear** (uma reta não explica os dados). No entanto, as variáveis ainda poderiam ter uma correlação *não-linear* (como uma curva parabólica). A letra D é falsa porque $r$ pode, sim, ser exatamente zero.

**4. Resposta Correta: B (A economia base estimada...)**
* **Por que?** Na equação $Y = aX + b$, o coeficiente $b$ (intercepto) é o valor de $Y$ quando $X$ é igual a zero. Trazendo para o cenário: se o usuário cadastrar zero metas ($X = 0$), a equação fica $Y = 50(0) + 15$, resultando em $Y = 15$. Já o valor 50 (coeficiente angular $a$) seria o quanto a economia aumenta por cada meta cadastrada (alternativa C).
